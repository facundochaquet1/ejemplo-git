# 🌍 EcoConciencia — Cuidemos el Medio Ambiente

Página web informativa que explica la importancia de cuidar el medio ambiente, los principales problemas que enfrenta el planeta y las acciones que todos podemos tomar para protegerlo.

## 📄 Descripción

Sitio estático hecho con **HTML y CSS**, con un diseño limpio, responsive y adaptado a dispositivos móviles. Utiliza emojis como iconos, por lo que no depende de imágenes externas.

## 🗂️ Estructura del proyecto

```
ejemplo-git/
├── index.html      # Estructura y contenido de la página
├── styles.css      # Estilos y diseño visual
└── README.md       # Este archivo
```

## ✨ Secciones

1. **Hero** — Título principal, navegación y botón de acción sobre un degradado verde.
2. **¿Por qué es tan importante?** — Tarjetas sobre agua limpia, aire puro y biodiversidad.
3. **Los problemas que enfrentamos** — Contaminación, cambio climático, deforestación y pérdida de especies.
4. **¿Qué podemos hacer?** — Acciones concretas para cuidar el planeta.
5. **Llamado a la acción** y pie de página.

## 🎨 Características técnicas

- Variables CSS (`:root`) para la paleta de colores.
- Diseño **responsive** con `grid` adaptable y *media queries*.
- Efectos *hover* en tarjetas, botones y acciones.
- Sin dependencias externas.

## 🚀 Cómo verla

Abre el archivo `index.html` con doble clic o desde tu navegador:

```bash
# O sirviéndola con un servidor local (opcional)
python -m http.server 8000
```

Luego visita `http://localhost:8000`.
